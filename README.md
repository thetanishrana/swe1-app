# SWE1 App

Django polls application with CI/CD pipeline.

## Features
- Continuous Integration with Travis CI
- Code coverage with Coveralls (82%)
- Automated testing
- Code quality checks with Black and Flake8
- Auto-deployment to AWS Elastic Beanstalk
